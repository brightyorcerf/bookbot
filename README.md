BookBot:
A real-world command-line application built in Python for text analysis. 
This tool processes entire novels to generate data reports on word and character frequency. 

Phase 1: Setup

Local Python environment configuration.
Git repository initialization for version control.
GitHub integration for remote project hosting.

Phase 2: Data Analysis
Reading large text files from the local filesystem.
Calculating total word counts using string manipulation.
Analyzing character frequency patterns using Python dictionaries.

Phase 3: The Report
Formatting raw dictionary data into a clean, human-readable terminal report.
Implementing sys.argv to allow analysis of any .txt file via the command line.


How to Run it

git clone https://github.com/brightyorcerf/bookbot.git
